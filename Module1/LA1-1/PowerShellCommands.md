## Terminal Commands
Note $ means terminal prompt
```bash
$ ls # list foled
$ cd <destination> # change directories
$ cd .. #.. parent directory
$ pwd #show present directory
$ rm # remove / delete file/directory
```
## File System
. # (one dot) present working directory
.. #(two dots) parent directory
